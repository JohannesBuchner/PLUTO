Detailed Description

Set the particle distribution to be either Uniformly spaced between x = 1.5 and 3.5 (configration #1, #2, #3 and #4), or placed in a Gaussian manner as defined by the function : Particles_SpaceDistrib

In addition to the above uniform distribution, configration #1 has particles being injected from the right X boundary (X1_END). The spectral evolution of particles in this configration #1 is switched off.

If the user defined constant "INIT_RANDOM_DISTRIBUTION" is set to YES, then the gaussian distribution of particles is set. (configration #5)

In case of runs with spectra (#2, #3, #4 and #5), each macro particle is assigned an initial power law spectra with a steep slope alpha = -9.0 between an energy interval of 0.01 E_0 and 1.0e4 E_0, where E_0 = 0.01 ergs (by default).
